# Visualize Frey face.

Display an image from the Frey faces dataset.

## Usage

``` r
show_frey_face(df, n, col = grDevices::gray(1/12:1))
```

## Arguments

- df:

  Data frame containing the Frey faces.

- n:

  Frame index of the image to display.

- col:

  List of colors to use in the display.
